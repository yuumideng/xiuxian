# UI 重新设计说明

## 设计理念

本次UI重新设计遵循以下核心原则：

### 1. 简化按钮样式（不超过三种）

**三种按钮类型：**
- **深色主按钮 (dark)**: `slate-700` 系列，用于主要操作
- **红色强调按钮 (red)**: `rose-600` 系列，用于重要/危险操作
- **灰色禁用按钮 (gray)**: `gray-400` 系列，用于次要操作或禁用状态

**按钮特性：**
- 保留并优化了点击下沉动画效果（`active:translateY(2px)`）
- 添加了悬停状态的阴影变化
- 统一的圆角和过渡效果
- 移除了老气的颜色，采用现代化的slate色系

### 2. 弹窗质感提升

**Modal组件改进：**
- 添加了装饰性背景渐变层
- 使用径向渐变创建光晕效果
- 背景采用毛玻璃效果（`backdrop-filter: blur(4px)`）
- 弹窗本身使用渐变背景增强层次感
- 优化了阴影系统，增加边框高光
- 改进了动画效果，使用更流畅的缓动函数

### 3. 配色方案优化

**主色调：**
- 背景：`slate-50` 到 `slate-100` 的渐变
- 卡片：白色到 `slate-50` 的渐变
- 文字：`slate-700` 系列（替代老气的灰色）
- 边框：`slate-200` 到 `slate-300`

**强调色：**
- 成功/增长：`emerald-600`（替代普通绿色）
- 警告/重要：`rose-600`（替代老气的红色）
- 信息/进度：`teal-400` 和 `cyan-400` 渐变
- 特殊：`amber-500`（金色按钮）

### 4. 细节优化

**阴影系统：**
- 卡片：`shadow-sm` 轻微阴影
- 悬停：`shadow-md` 中等阴影
- 弹窗：多层阴影组合

**圆角统一：**
- 小元素：`rounded-md` (0.5rem)
- 卡片：`rounded-lg` (0.75rem)
- 弹窗：`rounded-xl` (1rem)

**过渡动画：**
- 统一使用 `transition-all duration-150`
- 按钮点击下沉 2px
- 悬停时轻微上移 1px（部分元素）

## 改进的组件

### 1. GameButton.vue
- 简化为三种颜色方案
- 添加点击下沉动画
- 优化悬停和激活状态

### 2. Modal.vue
- 添加装饰性背景层
- 毛玻璃遮罩效果
- 渐变背景增强质感
- 优化动画曲线

### 3. BattleAttributeDetail.vue
- 更新配色为slate系列
- 优化公式框和计算框的渐变
- 改进滚动条样式

### 4. CreateCharacter.vue
- 背景渐变优化
- 输入框焦点状态增强
- 按钮统一为三种样式
- 优化卡片阴影和边框

### 5. SaveSelect.vue
- 存档卡片添加渐变背景
- 优化悬停和点击效果
- 进度条使用渐变和阴影
- 统一按钮样式

### 6. PlayerCard.vue
- 卡片添加渐变背景和边框
- 进度条使用双色渐变
- 优化按钮交互效果
- 统一配色方案

## 设计优势

1. **视觉统一**：所有组件使用相同的配色和样式系统
2. **现代感强**：采用渐变、阴影和毛玻璃等现代设计元素
3. **交互流畅**：统一的动画和过渡效果
4. **层次分明**：通过阴影和渐变建立清晰的视觉层次
5. **易于维护**：只有三种按钮样式，减少了复杂度

## 技术实现

- 使用 Tailwind CSS 的 slate 色系
- CSS 渐变和阴影组合
- Transform 动画实现按钮下沉
- Backdrop-filter 实现毛玻璃效果
- 统一的过渡时间和缓动函数
